# Manifest Templates

These templates document the expected schema without exposing any real metadata. All rows are synthetic examples.

- `slide_manifest_template.csv`: user-edited slide metadata input for ROI extraction
- `analysis_unit_manifest_template.csv`: ROI extraction output with manual QC fields
- `tile_manifest_template.csv`: tiling output manifest
- `manual_review_manifest_template.csv`: optional standalone review tracking sheet
- `split_manifest_template.csv`: optional downstream split sheet
